Command Line Cheat Sheet

Command Description
cp file.rb file2.rb copy file.rb into a new file file2.rb
cp file.rb foldername/ copy file.rb into folder called foldername
ln -s path/ link_name create a sym link to a folder
which program_name return the path of the program
history return list of recent commands. !number will run numbered command
mv filename.rb newfilename.rb move/rename a file
cat file streams the file to standard output
cat filename1.rb filename2.rb joins multiple files and streams the file to standard output
cat filename1.rb > filename2.rb streams content of filename1 and writes to filename2
ctrl + w delete the word on your left
ctrl + a take cursor to the start of the line
ctrl + e take cursor to the end of the line
!! repeat and run last command
echo prints to standard out
